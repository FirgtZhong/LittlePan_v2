## LittlePan_v2版本介绍
这是一个外链网盘管理系统，如果你有什么问题的话，欢迎去[Issues](https://github.com/FirgtZhong/LittlePan_v2/issues)进行详细说明<br>
## 食用说明
从[Releases](https://github.com/FirgtZhong/LittlePan_v2/releases)中下载最新的文件<br>
解压到网站根目录，打开网站，会自动转至安装界面<br>
按照操作提示即可<br>
